# Stopwatch Project

## Overview
This is a simple stopwatch application developed using HTML, CSS, and JavaScript. The project demonstrates basic DOM manipulation, event handling, and JavaScript timing functions.

## Features
- Start the stopwatch
- Pause the stopwatch
- Reset the stopwatch
- Time displayed in HH:MM:SS format
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Learning Outcomes
- DOM Manipulation
- Event Listeners
- setInterval() and clearInterval()
- JavaScript Functions
- Responsive Web Design
